TWITTER BOT

After working my way through a large portion of Udacity's Front End Development Nanodegree, I read an article regarding a twitter bot. I took to the Javascript I had learned and found a few tutorials and built @DonaldMeetAlice, a twitter bot that parses 3600 of Donald Trump's tweets with the book Alice in Wonderland using a Markov chain. I run it very little but it was a great learning experience.
